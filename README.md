# OpenThoughts
Open source React Native Mobile App for CBT mindfulness

### Installation
1. `yarn install`
1. `yarn start`

### Adding new IOS equivalnt packages
1. `cd ios`
1. `pod install`

### Linking new assets
1. `npx react-native link`

### Running for IOS
1. `yarn ios`

### Running for Android
1. `yarn android`

### Notes
1. Language - React Typescript
1. Frontend Library - UI Kitten
